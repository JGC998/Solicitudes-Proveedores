⚙ Cómo configurar la integración
  1. Abre el archivo .env en la raíz del proyecto
  2. Añade la URL de la API de la empresa:
    EXTERNAL_API_URL=https://api.tuempresa.com
  3. Añade el token de autenticación si lo requiere:
    EXTERNAL_API_KEY=tu-token-aqui
  4. Reinicia el servidor — sincronizará los productos automáticamente
  5. Consulta migracionAPI.md para adaptar el formato de los datos
